# devUnite
### MERN stack social network for developers

This application uses a stack like this:
 `MongoDB`,
 `Express.js`, 
 `React.js`, 
 `Node.js`.
 
Technologies that are present and used in this project: 
 `ES6+`, 
 `JWT`, 
 `Bcrypt`, 
 `Async/Await`,
 `Mongoose`,
 `React Hooks`,
 `Redux.js`,
 `Heroku & Git Deployment`.
